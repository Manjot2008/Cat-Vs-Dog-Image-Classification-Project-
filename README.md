# 🐶🐱 Cats vs Dogs Classification using CNN

This project is a Deep Learning-based image classification system that predicts whether an image contains a cat or a dog using a Convolutional Neural Network (CNN). The model is developed using TensorFlow and Keras and trained on a labeled dataset of cat and dog images.

The project demonstrates the complete workflow of a Computer Vision application, starting from image preprocessing and data augmentation to model training, evaluation, and prediction. Images are resized and normalized before being fed into the neural network. The CNN uses convolution and pooling layers to automatically learn important visual patterns and features from the dataset.

## The model performs binary classification:

0 → Cat 🐱

1 → Dog 🐶

## Key features of the project include:

CNN architecture for image recognition
Image preprocessing and normalization
Data augmentation for better generalization
Model training and validation
Prediction on custom images
Performance evaluation using accuracy metrics
